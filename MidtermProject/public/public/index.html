What is Kickspotter?

Kickspotter is a tool for exploring the data behind kickstarter projects, both finished and ongoing. Explore the successes and failures of the kickstarter platform!

Click on "Main" to explore the dataset with a bar chart visualizer. The data visualized can be modified with a set of controls on the left.

Adjust Categories to change the project category explored

Change the X Axis and Y Axis values to modify the arrangement and scaling of the data visualized

Besides the standalone values of Goal, Pledged, and Backers, there are also "Hype" Modifiers that indicate averages across their associated values.

The Backer Hype value represents the average pledge amount for total backers, while the Pledged Hype value correlates to the ratio above the project's goal that was reached.
